# Backend

Express + Prisma scaffold for the Job Fair platform.

## Scripts

- `pnpm dev`
- `pnpm build`
- `pnpm start`
- `pnpm typecheck`
- `pnpm prisma:generate`
- `pnpm db:push`

## Notes

- REST endpoints are stubbed to mirror the supplied API specification.
- Prisma models match the provided database schema.
- File uploads are expected under `uploads/` with resource-specific subfolders.
